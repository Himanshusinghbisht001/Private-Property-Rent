# 🏡 Private Property Rental

A modern Property Rental web application built with **React.js**, **React Router DOM**, and **Tailwind CSS**. The project provides a clean interface for browsing different types of rental properties with category-based navigation and reusable React components.

---

## 🚀 Features

- 🏠 Browse different rental properties
- 📂 Category-based navigation
- 🔥 Trending section
- 🏡 House listings
- 🌲 Cabins
- 🛏️ Rooms
- 🏕️ Tent Houses
- 🏊 Pool Houses
- 🚜 Farm Houses
- 🔍 Search bar UI
- 👤 Login & Signup pages
- 📋 List Your Home page
- 💬 Feedback page
- 🍔 Dropdown menu
- ⚡ Fast client-side routing
- 🎨 Modern UI built using Tailwind CSS

---

## 🛠️ Tech Stack

- React.js
- React Router DOM
- Tailwind CSS
- React Icons
- JavaScript (ES6)
